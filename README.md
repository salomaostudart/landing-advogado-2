# Studart & Associados — Landing Page Advocacia

Landing page moderna e responsiva para escritório de advocacia fictício, desenvolvida como projeto demonstrativo de portfolio.

## Visualizar

[https://salomaostudart.github.io/landing-advogado-2/](https://salomaostudart.github.io/landing-advogado-2/)

## Sobre

Projeto demonstrativo com tema claro e moderno (verde esmeralda + branco), contrastando com a [landing page Silva & Associados](https://salomaostudart.github.io/landing-advogado/) (tema escuro, azul-marinho + dourado). Juntas, demonstram versatilidade na criação de sites para o nicho jurídico.

## Funcionalidades

- Design responsivo (mobile, tablet, desktop)
- Navegação sticky com menu hamburger
- Hero com layout split (texto + imagem)
- Contadores animados na barra de confiança
- 6 áreas de atuação com ícones SVG
- Timeline "Como Funciona" em 4 etapas
- Seção de equipe com 4 advogados
- Depoimentos de clientes
- FAQ accordion interativo
- Formulário de contato
- Botão WhatsApp flutuante
- Animações ao scroll (IntersectionObserver)
- Acessibilidade (ARIA labels, skip link, foco visível)

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, media queries)
- JavaScript vanilla
- SVG inline para ícones
- Imagens via Unsplash CDN
